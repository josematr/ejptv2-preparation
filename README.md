Notas personales y cheatsheets basadas en el curso **"El Rincón del Hacker" (Mario)**.  
Material original y resumen por **Josema** — contenido creado por mí para repaso y práctica.

> ⚠️ **Importante:** No subo material con copyright del autor (vídeos/archivos de curso). Aquí solo hay resúmenes, comandos, scripts propios y ejercicios prácticos realizados por mí.

## Estructura
- `01-Networking/` — fundamentos, subredes, TCP/IP.
- `02-Enumeracion/` — herramientas y comandos para enumeración.
- `03-Scanning/` — nmap, escaneo activo/pasivo.
- `04-Explotacion-Host/` — técnicas en Linux y Windows.
- `05-Web/` — testing para aplicaciones web.
- `06-PostExplotacion/` — persistencia y exfiltración.
- `scripts/` — scripts propios para automatizar pruebas.
- `resources/` — enlaces, labs y prácticas.
